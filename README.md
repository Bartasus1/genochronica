# Genochronica

> Application to (primarily) self-host your family data, allowing you to organize and manage your family history.

---

## Features

- Ability to specify important events in person's life and add a biography
- Gallery of photos for every person and special events
- Option to export family members birthday to the calendar (planned)
- Option to create an account to make informations relative to you (planned)

Project is a part of Java course in Corporate Ready Certificate 2026 program.
